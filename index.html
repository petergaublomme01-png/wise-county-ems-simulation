How Optimal Staging Works

The optimizer evaluates every possible way to assign ambulances to staging sites and picks the configuration that best serves Wise County.

1
Generate combinations — tries all combinations with replacement from 6 staging sites for your fleet size. Multiple ambulances can share a station.
2
Score each configuration — runs the coverage model for each combination, measuring population-weighted 8-min coverage and worst-case response time across all 5 zones.
3
Select the best — picks the configuration that satisfies the 12-minute constraint while maximizing 8-minute population coverage.

Run times are near-instant — the coverage model is O(zones × ambulances) per configuration.
